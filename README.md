# dqwfun モンスターウォーカー（MonsterWalker）

ドラクエウォークのモンスター攻略情報を登録・共有するファンツールアプリです。

## 概要

- モンスターの攻略編成（武器・職業・行動順序）を登録・共有
- 自分の保有武器に合った攻略情報を検索
- いいね・お気に入り機能で便利な攻略情報を管理

## 主な機能

| 機能 | 説明 |
|------|------|
| 攻略情報検索 | モンスター・武器で検索、いいね数順ソート |
| 攻略情報登録 | パーティ構成（4人分）、行動順序、攻略タイプを登録 |
| いいね | 役立った攻略情報を評価 |
| お気に入り | 検索条件・攻略情報を保存 |
| ランキング | いいね数ランキング、登録者ランキング |

## 技術スタック

- **フロントエンド**: React Native + Expo
- **バックエンド**: Supabase (PostgreSQL + Auth + Storage)
- **プラットフォーム**: iOS / Android
